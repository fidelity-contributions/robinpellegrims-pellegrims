# @pellegrims/nx-remotecache-s3

> [!WARNING] > **DEPRECATED:** This project is no longer maintained. Users are encouraged to migrate to [Nx Powerpack](https://nx.dev/powerpack/remote-cache) or other community-maintained remote cache runners.

This repository contains the [@pellegrims/nx-remotecache-s3](./libs/nx-remotecache-s3) package, which is a task runner for [Nx](https://nx.dev) that uses S3 storage as a remote cache.

## Status

This project is now in maintenance mode and is deprecated due to its reliance on the unmaintained `nx-remotecache-custom` package.
